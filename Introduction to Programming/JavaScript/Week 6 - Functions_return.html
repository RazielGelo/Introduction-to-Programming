JavaScript Functions 3 with return

This section tests you to use the return keyword with functions in order to write functions that return a value. In order to test your functions out that return values (instead of console.logging values), open the console window (shift + ctrl + i), type the name of the function and include parameters if required. This should output the result directly in the console if successful.

  1. Doctor
    Write a function that takes in two parameters (firstName and lastName) which are strings. return the full name with the "Dr." title placed in front.
  2. Doctor EXTENDED
    Using the function you made above, modify the console.log below so that the greeting shows your name with the Dr. prefix.
  3. Area of Rectangle
    Write a function that takes in two parameters (width and height) which are numbers representing the width and height of a rectangle. return the area of the rectangle.
  4. Perimeter of Rectangle
    Write a function that takes in two parameters (width and height) which are numbers representing the width and height of a rectangle. return the perimeter of the rectangle. This is length of the rectangle multiplied by 2, plus the height of the rectangle multiplied by 2.
    Use default parameter values of 0 in case user forgets to include values for parameters.
  5. Using Other Functions inside of Function
    Create a single function that takes in two parameters (width and height) which are numbers representing the width and height of a rectangle. In this function, make use of the previous two functions which returns the area and perimeter of a rectangle. In a single console log output "Height = X, Width = Y, Area = Z, Perimeter = W". Use default values of 0 for the parameters in this function.
  6. Break String Into Array
    Create a function that takes in a string as a parameter, uppercases all the letters and returns an array with each uppercase letter being an item in that array.
    You should check out the JavaScript string method called split() to help you in this regard. Some test strings have been provided
  7. String Vowel Checker
    Create a function that takes in a single string parameter. First check to make sure that the parameter is a string. If it is not a string return null. You may want to look up typeof in JavaScript to help you here. If it is a string check to see if the first letter is a vowel. If it is return true. If it isn't return false.
  8. String Vowel Checker EXTENDED
    Create a new function that doesn't take in a parameter. This function should ask the user what their name is using a prompt. Store their name in a variable and then use the function you created in the previous question to check if the name starts with a vowel. Console.log the following based on the returned value: "Your names starts with a vowel" or "Your name starts with a consonant" or "Your name isn't a string".
  9. Last Item in Array
    Create a function that takes in an array as a parameter. return the last element inside the array
  10. Modify Array
    Create a function that takes in a single parameter that is an array containing strings. In this function, return a new array that contains all the strings uppercased.
  11. Filter Array
    Create a function that takes in one parameter that is an array containing strings. In this function, return a new array that contains only items from the original array that start with a capital letter. Make use of a function you already created to help you here.
  12. Count the Number of Words in Sentence that Begin with Vowel
    You are given a string variable which is a sentence containing a bunch of words. Use the functions that you have created above to console.log the following: "There are X words in the sentence that begin with a vowel". You may want to look up the JavaScript method called split() to help you in this regard.
  13. Count the Votes and return an Array
    Create a function that takes in a single array as a parameter. An array called voteArr has been provided to test. In the function, count the number of "Yes" and "No" votes inside of the array. return an array that contains the number of Yes votes as the first item and the number of No votes as the second item in this returned array.
  14. Number Function
    Create a separate function for each of the following. These functions must take in a single parameter which is an array:
    • Checks to see if all items in array are numbers
    • returns the sum of all numbers in array
    • returns the average of all numbers in array
    • returns the largest number in array
    Use all of these functions to console.log

    "There are X numbers in this array.
    The total sum of all numbers is Y.
    The average of all numbers is Z.
    The largest number in the array is W"
