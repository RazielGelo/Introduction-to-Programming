Pair Programming

Pair programming is a software development technique in which two programmers work together at the same workstation on the same project. One, the driver, writes code while the other, the observer or navigator, reviews each line of code as it is typed in. The two programmers switch roles frequently. Talking, discussing and explaining are highly encouraged.

  1. Number Functions
    Write code and use functions that take in a single array and checks to see that all items are numbers. If they are, your code should produce the following console.log:
    "There are X numbers in this array. The total sum of all numbers is Y. The average of all numbers is Z. The largest number in the array is W"

    If the array contains an item that is not a number, your code should just console.log "ERROR".

    Try to use multiple functions in your code to have it organised better. For instance one function could calculate and return the sum of the array, another code check to see if the array contains numbers and return true or false etc. Then one main function could call these to perform the program.
